Batman

Algorithm

The algorithm takes as input an unweighted graph and the id of the source vertex s. The input graph can be directed or undirected, it does not matter to the algorithm 
    vector> adj;  // adjacency list representation
        int n; // number of nodes
        int s; // source vertex
        
        queue q;
        vector used(n);
        vector d(n), p(n);
        
        q.push(s);
        used[s] = true;
        p[s] = -1;
        while (!q.empty()) {
            int v = q.front();
            q.pop();
            for (int u : adj[v]) {
                if (!used[u]) {
                    used[u] = true;
                    q.push(u);
                    d[u] = d[v] + 1;
                    p[u] = v;
                }
            }
        }
